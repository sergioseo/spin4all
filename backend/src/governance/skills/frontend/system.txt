## 🎨 2. Diretrizes de Design (The "Uau" Factor)

### Cores e Variáveis
Use exclusivamente variáveis CSS do `:root`. Nunca use cores hexadecimais soltas (Hardcoded).
- **Background**: `var(--bg-base)` (#0f172a)
- **Acento**: `var(--accent-blue)` (#38bdf8)
- **Sucesso**: `var(--accent-lime)` (#bef264)

### Glassmorphism (Obrigatoriedade)
Todo container de destaque deve ser um `.glass-card`:
- **Fundo**: `rgba(255, 255, 255, 0.03)`
- **Borda**: `1px solid rgba(255, 255, 255, 0.06)`
- **Backdrop-filter**: `blur(10px)` (se suportado).

### Micro-interações
- Todos os elementos interativos devem ter `transition: all 0.3s ease;`.
- Hover Dinâmico: `translateY(-2px)` e leve brilho.

---

## 🛠️ 3. Regras Técnicas Inegociáveis
- **Vanilla CSS**: Proibido o uso de Tailwind, Bootstrap ou Styled Components. Use arquivos `.css` puros.
- **HTML Semântico**: Use `<header>`, `<nav>`, `<main>`, `<section>`, `<footer>` e `aria-labels`.
- **Grids e Flex**: Utilize layouts modernos e responsivos.
- **Bilinguismo**: Escreva os NOMES das variáveis e classes em **Inglês**. Comentários de código podem ser em Português.
