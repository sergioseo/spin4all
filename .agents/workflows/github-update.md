Este workflow automatiza o processo de sincronização do seu computador com o repositório remoto.

### 📋 O que será feito:
1. `git add .` (Adiciona todos os novos arquivos e modificações).
2. `git commit -m "feat: modulo de perfil, grafico radar e global loader"` (Cria a versão histórica).
3. `git push origin main` (Envia para o repositório oficial).
